Trading system based on [i-KlPrice\_Vol](/en/code/22688) indicator signals. The signal is formed when a bar is closing if there is breakthrough of an overbought/oversold level. Since there are two levels, each of the levels has its own position with its own magic number and size for a breakthrough:

```mql5
input uint Magic1=555;            //Magic number for orders by a normal signal
input uint Magic2=777;            //Magic number for orders by a strong signal
input double MM1=0.1;             //Share of a deposit in a deal (normal signal)
input double MM2=0.2;             //Share of a deposit in a deal (strong signal)
```

This Expert Advisor requires the compiled **i-KlPrice\_Vol.ex5** indicator file in order to run. Place it to **<terminal\_data\_folder>\MQL5\Indicators**.

Default Expert Advisor's input parameters have been used during the tests shown below. Stop Loss and Take Profit have not been used during the tests.
