The [MomentumCloud](/en/code/13301) indicator with the timeframe selection option available in the input parameters:

```mql5
input ENUM_TIMEFRAMES TimeFrame=PERIOD_H4;  // Indicator chart period (timeframe)
```

The indicator requires MomentumCloud.mq5 indicator file. Place it in <terminal\_data\_folder>\MQL5\Indicators.
