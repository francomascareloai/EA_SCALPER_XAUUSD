This indicator is the simplified version of [Monitoring-Spread](https://www.mql5.com/en/code/9324) tool.

It allows to use the reversed symbols (**1 / X**) in a realtime.

### Image:

![](https://c.mql5.com/18/21/ReverseSymbol.png "Иллюстрация работы индикатора ReverseSymbol")  
ReverseSymbol example

**How to use:**

* Attach indicator to the chart, set "**allow DLL use**".
* Open chart of "reversed" symbol in **Offline** mode (File->Open Offline).
