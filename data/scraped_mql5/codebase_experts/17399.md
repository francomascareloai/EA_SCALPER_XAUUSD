The [XprofuterDD](/en/code/1024) indicator with the timeframe selection option available in the input parameters.

```mql5
input ENUM_TIMEFRAMES TimeFrame=PERIOD_H4;  // Indicator chart period (timeframe)
```

To provide correct operation of the indicator, place the compiled XprofuterDD.mq5 indicator file in terminal\_data\_folder\MQL5\Indicators.
