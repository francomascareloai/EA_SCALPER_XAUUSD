The trading system based on the signals of the [TimeZonePivotsOpenSystem](/en/code/22407/202302#!tab=code "Rj_SlidingRangeRj_Digit_System - indicator for MetaTrader 5") indicator. The signal is formed when a bar is closing if there is a channel breakthrough.

For the generated EA to operate correctly, the compiled **TimeZonePivotsOpenSystem.ex5** indicator file should be available in the <terminal\_data\_directory>\MQL5\Indicators folder.

The EA works on timeframes not higher than H1:

```mql5
input ENUM_TIMEFRAMES InpInd_Timeframe=PERIOD_H1; // TimeZonePivotsOpenSystem indicator timeframe
```
