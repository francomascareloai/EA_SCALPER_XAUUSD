The [i-CAi](/en/code/15969) indicator displays the most recent value as a price label with the the possibility to round the channel levels up to a required number of digits. The number of digits to round is set in the input variable *Digit:*

```mql5
input uint Digit=3; //Number of digits to round to
```

The indicator uses SmoothAlgorithms.mqh library classes (copy it to <terminal\_data\_folder>\MQL5\Include). The use of the classes was thoroughly described in the article ["Averaging Price Series for Intermediate Calculations Without Using Additional Buffers"](/en/articles/180).
