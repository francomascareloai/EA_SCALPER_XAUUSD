# State Checking

Functions that return parameters of the current state of the client terminal

| Function | Action |
| --- | --- |
| [GetLastError](/en/docs/check/getlasterror) | Returns the last error |
| [IsStopped](/en/docs/check/isstopped) | Returns true, if an mql5 program has been commanded to stop its operation |
| [UninitializeReason](/en/docs/check/uninitializereason) | Returns the code of the reason for deinitialization |
| [TerminalInfoInteger](/en/docs/check/terminalinfointeger) | Returns an integer value of a corresponding property of the mql5 program environment |
| [TerminalInfoDouble](/en/docs/check/terminalinfodouble) | Returns a double value of a corresponding property of the mql5 program environment |
| [TerminalInfoString](/en/docs/check/terminalinfostring) | Returns a string value of a corresponding property of the mql5 program environment |
| [MQLInfoInteger](/en/docs/check/mqlinfointeger) | Returns an integer value of a corresponding property of a running mql5 program |
| [MQLInfoString](/en/docs/check/mqlinfostring) | Returns a string value of a corresponding property of a running mql5 program |
| [Symbol](/en/docs/check/symbol) | Returns the name of a symbol of the current chart |
| [Period](/en/docs/check/period) | Returns the current chart timeframe |
| [Digits](/en/docs/check/digits) | Returns the number of decimal digits determining the accuracy of the price value of the current chart symbol |
| [Point](/en/docs/check/point) | Returns the point size of the current symbol in the quote currency |
