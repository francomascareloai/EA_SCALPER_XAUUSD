# Trade Signals

This is the group of functions intended for managing trade signals. The functions allow:

* get information about trade signals, available for copying,
* get and set the signal copy settings,
* subscribe and unsubscribe to the signal copying using MQL5 language functions.

| Function | Action |
| --- | --- |
| [SignalBaseGetDouble](/en/docs/signals/signalbasegetdouble) | Returns the value of double type property for selected signal |
| [SignalBaseGetInteger](/en/docs/signals/signalbasegetinteger) | Returns the value of integer type property for selected signal |
| [SignalBaseGetString](/en/docs/signals/signalbasegetstring) | Returns the value of string type property for selected signal |
| [SignalBaseSelect](/en/docs/signals/signalbaseselect) | Selects a signal from signals, available in terminal for further working with it |
| [SignalBaseTotal](/en/docs/signals/signalbasetotal) | Returns the total amount of signals, available in terminal |
| [SignalInfoGetDouble](/en/docs/signals/signalinfogetdouble) | Returns the value of double type property of signal copy settings |
| [SignalInfoGetInteger](/en/docs/signals/signalinfogetinteger) | Returns the value of integer type property of signal copy settings |
| [SignalInfoGetString](/en/docs/signals/signalinfogetstring) | Returns the value of string type property of signal copy settings |
| [SignalInfoSetDouble](/en/docs/signals/signalinfosetdouble) | Sets the value of double type property of signal copy settings |
| [SignalInfoSetInteger](/en/docs/signals/signalinfosetinteger) | Sets the value of integer type property of signal copy settings |
| [SignalSubscribe](/en/docs/signals/signalsubscribe) | Subscribes to the trading signal |
| [SignalUnsubscribe](/en/docs/signals/signalunsubscribe) | Cancels subscription |
