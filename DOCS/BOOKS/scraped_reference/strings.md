# String Functions

This is a group of functions intended for working with data of the [string](/en/docs/basis/types/stringconst) type.

| Function | Action |
| --- | --- |
| [StringAdd](/en/docs/strings/stringadd) | Adds a string to the end of another string |
| [StringBufferLen](/en/docs/strings/stringbufferlen) | Returns the size of buffer allocated for the string |
| [StringCompare](/en/docs/strings/stringcompare) | Compares two strings and returns 1 if the first string is greater than the second; 0 - if the strings are equal; -1 (minus 1) - if the first string is less than the second one |
| [StringConcatenate](/en/docs/strings/stringconcatenate) | Forms a string of parameters passed |
| [StringFill](/en/docs/strings/stringfill) | Fills out a specified string by selected symbols |
| [StringFind](/en/docs/strings/stringfind) | Search for a substring in a string |
| [StringGetCharacter](/en/docs/strings/stringgetcharacter) | Returns the value of a number located in the specified string position |
| [StringInit](/en/docs/strings/stringinit) | Initializes string by specified symbols and provides the specified string length |
| [StringLen](/en/docs/strings/stringlen) | Returns the number of symbols in a string |
| [StringSetLength](/en/docs/strings/stringsetlength) | Sets a specified length (in characters) for a string |
| [StringReplace](/en/docs/strings/stringreplace) | Replaces all the found substrings of a string by a set sequence of symbols |
| [StringReserve](/en/docs/strings/stringreserve) | Reserves the buffer of a specified size for a string in memory. |
| [StringSetCharacter](/en/docs/strings/stringsetcharacter) | Returns a copy of a string with a changed value of a symbol in a specified position |
| [StringSplit](/en/docs/strings/stringsplit) | Gets substrings by a specified separator from the specified string, returns the number of substrings obtained |
| [StringSubstr](/en/docs/strings/stringsubstr) | Extracts a substring from a text string starting from a specified position |
| [StringToLower](/en/docs/strings/stringtolower) | Transforms all symbols of a selected string to lowercase |
| [StringToUpper](/en/docs/strings/stringtoupper) | Transforms all symbols of a selected string into capitals |
| [StringTrimLeft](/en/docs/strings/stringtrimleft) | Cuts line feed characters, spaces and tabs in the left part of the string |
| [StringTrimRight](/en/docs/strings/stringtrimright) | Cuts line feed characters, spaces and tabs in the right part of the string |
