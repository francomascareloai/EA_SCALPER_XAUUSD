**Library of functions for working with strings.**

The following functions are available:

* **StringToArray** (3 overloads): extracting of the same type elements from the string into an array of a necessary type (the StringSplit analog);
* **StringToPeriod** and **PeriodToString**: conversion of time frame names in tabulation at ENUM\_TIMEFRAMES and back.

The library will refill as often as required.
