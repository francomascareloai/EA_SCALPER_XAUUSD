The Exp\_2pbIdealXOSMA Expert Advisor is drawn on the basis of [2pbIdealXOSMA](/en/code/1164) indicator. A signal to perform a deal is formed at bar closing if the histogram changes the direction.

Place 2pbIdealXOSMA.ex5 compiled file to the terminal\_data\_folder\MQL5\Indicators.

Default Expert Advisor's input parameters have been used during the tests shown below. Stop Loss and Take Profit have not been used during the tests.
