The Expert Advisor shows the current values of World Stock Indices from [Google Finance](https://www.mql5.com/go?link=https://www.google.com/finance "https://www.googe.com/finance") using [WebRequest](https://docs.mql4.com/common/webrequest) function. The quotes are updated every 10 seconds.

For working with server you need to add "**https://www.google.com/finance**" to the list of the allowed URLs (Main menu->Tools->Options, "Expert Advisors" tab):
