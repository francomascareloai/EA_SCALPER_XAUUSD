The [Fractional\_Bands](/en/code/17074) indicator with the timeframe selection option available in input parameters:

```mql5
input ENUM_TIMEFRAMES TimeFrame=PERIOD_H4;  // Indicator chart period (timeframe)
```

The Fractional\_Bands\_HTF repeater indicator requires the compiled custom indicator file Fractional\_Bands.mq5 in order to compile. Place it in *<terminal\_data\_folder>\MQL5\Indicators*.

After compilation, the Fractional\_Bands\_HTF.ex5 indicator file contains the Fractional\_Bands.ex5 indicator as a [resource](/en/docs/runtime/resources), and therefore, it is not required to be present in the terminal folder for the compiled indicator to work! For this purpose, the corresponding code has been added to the indicator code in order to include the Fractional\_Bands indicator in the executable file.

The indicator executable file has been added as resources at the global scope

//---- Include custom indicators in the indicator code as resources  
#resource \\Indicators\\Fractional\_Bands.ex5

Changed the string path to the indicator used as resource in the block of the OnInit() function

//---- getting handle of the Fractional\_Bands indicator  
   Ind\_Handle=iCustom(Symbol\_,TimeFrame,"::Indicators\\Fractional\_Bands",  
                      e\_period,normal\_speed,PIP\_Convertor,alpha,IPC,0,PriceShift,Upper\_color,Middle\_color,Lower\_color,FontSize);

Thus, the compiled executable file of the repeater indicator can be used on other trade terminals on its own without the original indicator.
