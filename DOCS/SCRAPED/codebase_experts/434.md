In contrast with the standard [MACD histogram](https://www.metatrader5.com/en/terminal/help/indicators/oscillators/macd "https://www.metatrader5.com/en/terminal/help/indicators/oscillators/macd"), the JMA smoothing is used in all three averagings.

It can be very useful, especially when working with highly volatile currency crosses.

The indicator uses CJJMA class of the SmoothAlgorithms.mqh library. The use of this class is described in the article ["Averaging Price Series for Intermediate Calculations Without Using Additional Buffers"](/en/articles/180).

* The smoothalgorithms.mqh library must be placed to the **terminal\_data\_folder**\MQL5\Include.
* The jmacd.mq5 indicator must be placed to the **terminal\_data\_folder**\MQL5\Indicators.
