[A five-fold exponential moving average](/en/code/945) that displays the most recent value as a price label with the possibility to round the indicator levels to the required number of orders The number of digits to round is set in the input variable *Digit:*

```mql5
input uint Digit=3; //Number of digits to round to
```

The indicator uses the classes of library SmoothAlgorithms.mqh (to be copied to <terminal\_data\_folder>\MQL5\Include). The use of the classes is described in details in article [Averaging Price Series for Intermediate Calculations without Using Additional Buffers](/en/articles/180).
