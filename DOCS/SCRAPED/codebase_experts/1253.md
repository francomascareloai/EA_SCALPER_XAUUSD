Trading system using the [GG-RSI-CCI](/en/code/1243) signal indicator. A signal to perform a deal is formed at bar closing when the bar changes color from red to green or blue. Exit from the deals is performed either by a pending order, or by bar color changing.

Place GG-RSI-CCI.ex5 compiled file to the terminal\_data\_folder\MQL5\Indicators.

Default Expert Advisor's input parameters have been used during the tests shown below. Stop Loss and Take Profit have not been used during the tests.
