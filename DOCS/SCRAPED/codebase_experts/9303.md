Indicator shows vertical lines at last week bar.

![](https://c.mql5.com/18/21/friday_eng_3.png)

**Parameters:**

**Counted\_Bars** - number of bars in the calculation, 0 means all bars.

**type\_line** - line type.

**Color** - line color.

Indicator runs once, to run again make reinitialization.

It deletes only its own lines, the lines are names using the bar dates.

[![](https://c.mql5.com/18/21/friday_pic_2.png)](https://c.mql5.com/18/21/friday_pic.png)
