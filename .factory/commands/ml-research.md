# /ml-research - ML Trading Research

Conduct comprehensive research on Machine Learning applications for algorithmic trading.

## Activation

Load and execute the **ml-trading-research** skill from:
`.factory/skills/ml-trading-research.md`

## Research Topics

### Model Architectures
- LSTM / xLSTM for price prediction
- Transformers for time series
- CNN for pattern recognition
- GRU for volatility forecasting
- Ensemble methods

### Integration
- ONNX + MQL5 best practices
- Real-time inference optimization
- Feature normalization for production

### Statistical Methods
- Hurst exponent for regime detection
- Shannon entropy for noise filtering
- Kalman filters for trend estimation
- Hidden Markov Models for regime switching

### Validation
- Walk-Forward Analysis
- Monte Carlo simulation
- Cross-validation for time series
- Overfitting detection

## Usage

```
/ml-research LSTM gold price prediction accuracy
/ml-research ONNX MQL5 integration tutorial
/ml-research Hurst exponent trading implementation
/ml-research feature engineering financial time series
```

## Output Format

Research results include:
- Academic evidence with sources
- Implementation resources (repos, docs)
- Performance data and benchmarks
- Specific recommendations
- Risk assessment
- Confidence levels

## Tools Used

- `perplexity-search` - AI-synthesized research
- `brave-search` - Broad web search
- `github___search_repositories` - Code repos
- `context7___get-library-docs` - Technical docs
